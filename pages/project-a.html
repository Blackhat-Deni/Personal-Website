Another milestone in my journey of building web applications.
The Cinema Ticket Booking System was born from a simple frustration. While spending time with friends, we realized that the only movie hall in our hometown didn't offer online ticket booking. Customers had to be physically present to secure seats for their favorite movies.
In today's digital age, that limitation felt outdated. So I decided to build a modern web application that allows users to browse movies and book tickets online.
This project represents both problem-solving and practical implementation of my growing development skills.
For the source code, visit my GitHub.

For our summer university project, Dipesh Regmi, Suman Baduwal, and I developed a Library Management System.
The application enables efficient management of library operations, including adding, updating, deleting, and searching for books. It also manages student records and maintains borrowing history for proper tracking and accountability.
While we identified certain security vulnerabilities that need improvement, the system performs smoothly in terms of functionality. Database design and implementation were particularly engaging aspects of this project.
For the complete source code, please visit my GitHub.

So, this is it — my personal website.
I actually got this idea from the book Show Your Work by Austin Kleon. But I want more from it. There are two reasons for this.
First, I want a separate space on the internet just for me, where I can do anything I want — all my imagination and execution. I have a problem with Meta apps because I feel like I can't customize them the way I want. Here, I can do anything I want. Upload anything I want with zero restrictions. I'm in charge here.
Secondly, since I'm still honing my skills in the cybersecurity sector, I wanted to test my skills and knowledge. I plan to perform security attacks on my own website and then upgrade it accordingly, making it more secure against external attacks.
It's like killing a snake without breaking the stick.
I decided to build this website following a versioning system, which you can see at the end of the homepage. I structured the website in a way that allows me to upgrade it easily.
Version 1 (1-5) will focus on the basics.
Version 2 (1-5) will focus on security vulnerabilities and optimization.
Long journey? Yes, it is. But it's also fun learning new things.
